カラーコード変換

色を選ぶと、そのHEXコードを表示します。

HEX: #1b2342
<input type="color" id="cp" value="#1b2342">
<div id="sw"></div>
<span id="hex">#1b2342</span>
<script>
  cp.addEventListener('input', e => {
    sw.style.background = e.target.value;
    hex.textContent = e.target.value;
  });
</script>

文字数カウンター

入力した文章の文字数をリアルタイムで数えます。

文字数: 0
<textarea id="cc"></textarea>
<span id="num">0</span>
<script>
  cc.addEventListener('input', e => {
    num.textContent = e.target.value.length;
  });
</script>
